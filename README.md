# Messengr

API WebSocket desenvolvida com NestJS e Socket.IO, responsável pelo gerenciamento de salas privadas, conexões simultâneas, troca de mensagens em tempo real e controle de eventos como entrada, saída e digitação de usuários. A aplicação garante isolamento por sala, limite de participantes e comunicação eficiente entre clientes de forma escalável e organizada.

## Como rodar o projeto

Utilizando npm

```bash
npm install
npm run start
```
